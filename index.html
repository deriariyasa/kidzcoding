Tentang Kami

Kidzcoding merupakan platform belajar coding untuk anak. Pembelajaran di Kidzcoding dapat dilakukan dengan kelas belajar online sehingga dapat di akses dimana saja dan kapan saja. Program belajar di Kidzcoding akan men-challenge anak untuk bisa mengembangkan problem solving skill, kreativitas, logika dan kemampuan matematika.

Kurikulum kami menggabungkan cara pengajaran progressive teaching methods dan online activities yang menyenangkan.

Daftar Pelajaran

Junior Class (Usia 6 - 8 tahun)

Di kelas ini anak akan belajar membuat game sederhana dengan Scrath. Scratch adalah bahasa pemrograman visual yang interaktif dan mengasyikan untuk belajar coding bagi anak.

Kids Class (Usia 8 - 12 tahun)

Pada kelas ini anak akan belajar bagaimana mebangun website dari awal mulai dari tahap sketsa, desain layout, hingga mengetikkan code HTML, CSS dan JavaScript

Lihat kelas

Teen Class (Usia 13 - 17)

Di kelas ini peserta didik akan diajarkan untuk menjadi seorang penemu atau seorang technopreneur di bidang teknologi, terutama teknologi automasi dan robotika.
